# FarmStand
<h1>농산품 판매대 WEB </h1>
<ul>
  <li>NodeJS module: express,mongoose</li>
  <li>Javascript Template: ejs,ejs-mate</li>
  <li>CSS : Bootstrap v5</li>
  <li>Database: MongoDB V5.0.6</li>
  <li>Editor: VS Code </li>
</ul>
<p>
  [주요기능]
  1. 모든 품목 조회 기능 
  2. 선택한 품목 상세 조회 기능
  3. 새로운 품목 추가기능
  4. 기존 품목 업데이트
  5. 선택한 품목 삭제 기능
</p>
