# FarmStand
## 농산품 판매대 WEB 
* 사용기술
  + NodeJS : express,mongoose
  + Template: ejs,ejs-mate
  + CSS : Bootstrap v5
  + Database: MongoDB V5.0.6
  + Editor: VS Code 

<hr/> 
  [주요기능]
<li> 모든 품목 조회 기능
<li> 선택한 품목 상세 조회 기능
<li> 새로운 품목 추가기능
<li> 기존 품목 업데이트
<li> 선택한 품목 삭제 기능
